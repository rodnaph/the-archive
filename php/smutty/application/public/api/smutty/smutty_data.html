Smutty.Smutty_Data

Class Smutty_Data

Smutty_Object
  |
  +--Smutty_Data

public class Smutty_Data
extends Smutty_Object

this class controls access to GET, POST and SERVER data. it's getInstance() method will return an instance with the correct type for the request method (GET/POST), but you can use this object to control access to any kind of data really.
